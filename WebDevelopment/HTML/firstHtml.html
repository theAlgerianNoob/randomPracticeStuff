How to make lists

To make an ordered list:

  1. Write "ol"
  2. Then write "li" below it
  3. Then write what you want to make as a list

To make an unordered list:

  • Write "ul"
  • Then write "li" again below it, similar to the previous method
  • Then, again, write what you want to make as a list

How to make tables

To make a table you have to do the following:

  1. Define the table by writing "table"
  2. Then divide the table in rows using "tr"
  3. Then divid the rows into columns by using "td"

Block level and inline elements

Block level elements always starts from a new line, such as headers "h1", paragraphs "p" etc

Inline elements are normally displayed without line breaks, for example, writing strong texts using "strong", or bold texts using "b" etc

The "div" element is a block level element that is often used as a container for other HTML elements.

There are different types of elements that can be used as block level or inline elements:

APPLET Embedded Java applet
INFRAME Inline Frame

MAP Image map
OBJECT Embedded Object
SCRIPT Script within an HTML document

Bare in mind that you can insert inline elements inside block elements.
