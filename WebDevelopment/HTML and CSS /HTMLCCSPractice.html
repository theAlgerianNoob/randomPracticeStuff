Working With Texts

Font-Family

There are two types of font family names:

  1. Font-Family
  2. Generic-Family

The font-family is a specific font family like new times roman or arial.

The generic-family is a group of font families with a similar look like serif or monospace.

The following texts are written with generic family and font family fonts:

  • This paragraph is written in Serif

  • This paragraph is written in Sans-Serif

  • The paragraph is written in Monospace

  • This paragraph is written in Cursive

  • This paragraph is written in Fantasy

It is good practice to write multiple fonts incase the browser does not support the font you specified. You can seperate each font with commas.

Font-Size

Not only is there different font types, but there are also font sizes that
can be adjusted. There are key words that are used to adjust them:

  • xx-small

  • x-small

  • small

  • medium

  • large

  • x-large

  • xx-large

Font sizes can be changed using pixels and ems. ems is another way
to manipulate the size of the texts. All major browsers will resize the texts
for you. If you havent already specified the font size, the em can be calculated.
Knowing that the default browser size is 16px the following can calculate the em:

em=pixels/16

Font-Style

There are three main values for font-style properties:

  • Normal

  • Italic

  • Oblique

Oblique is very similar to italic, but less suported

Font-Weight

Similar to font styles, there are three additional values for the wieght of the font, excluding the normal value. Here are all the font weights:

  • Normal

  • Lighter

  • Bold

  • Bolder

The font weight can also be defined with numbers
ranging from 100 (Thin) to 900 (thick). 400 is
similar to normal and 700 is similar to bold.

Font-Variant

The font variant property allows you to convert the font into small caps.
The values can be set as:

  • Normal
