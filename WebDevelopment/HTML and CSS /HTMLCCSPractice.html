Working With Texts

Font-Family

There are two types of font family names:

  1. Font-Family
  2. Generic-Family

The font-family is a specific font family like new times roman or arial.

The generic-family is a group of font families with a similar look like serif or monospace.

The following texts are written with generic family and font family fonts:

  • This paragraph is written in Serif

  • This paragraph is written in Sans-Serif

  • The paragraph is written in Monospace

  • This paragraph is written in Cursive

  • This paragraph is written in Fantasy

It is good practice to write multiple fonts incase the browser does not support the font you specified. You can seperate each font with commas.
