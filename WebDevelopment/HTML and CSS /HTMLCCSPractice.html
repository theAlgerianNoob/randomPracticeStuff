Font-Family

There are two types of font family names:

  1. Font-Family
  2. Generic-Family

The font-family is a specific font family like new times roman or arial.

The generic-family is a group of font families with a similar look like serif or monospace.

White Spacing

White space property specifies how white space inside and element is handled. There are four values:

nowarp Puts the whole text onto one line and colapses all the whitespace into single whitespace.
pre Text will only wrap on line breaks and white space.
pre-line Text will wrap where there is a break in code, but extra white space is ignored.
pre-wrap Text will wrap when necessary, and on line breaks

Styling Lists

Knowing that there are two different types of lists, ordered and unordered, we can also style these lists further making them more appealing. By using the list-style-type function, we can have a wide range of lists styles. For example here are a few:

  1. Lower-alpha - using lowercase alphabets instead of numbers.
  • Circle - using circular bulletpints that arent filled.
  • Square - instead of circular bullet points we use squares.

You can also use two other style types properties:

  • List-style-image :- Here you can use an image instead of a bulletpoint
  • List-style-position :- This specifies the position of the marker box, it can be either inside or outside.

However, There is a way by simply using all these properties by simply using one line. And that is by using the list-style property. By using the list-style property you can declare:

  • list-style-type
  • list-style-position
  • list-style-image

All in one line by simply using the list-style property.

Table layout is a function specifies the how width of the table columns is calculated. There are two values:

  • Auto :- when column or cell width is not specified, the column width will be in proportion to the amount of content in the cells that make up the column.
  • Fixed :- When column or cell width is not specified, the column width will not be affected by the amount of content in the cells that make up the column.
