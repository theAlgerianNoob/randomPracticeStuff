Working With Texts

Font-Family

There are two types of font family names:

  1. Font-Family
  2. Generic-Family

The font-family is a specific font family like new times roman or arial.

The generic-family is a group of font families with a similar look like serif or monospace.

The following texts are written with generic family and font family fonts:

  • This paragraph is written in Sans-Serif

  • The paragraph is written in Monospace

  • This paragraph is written in Fantasy

It is good practice to write multiple fonts incase the browser does not support the font you specified. You can seperate each font with commas.

Font-Size

Not only is there different font types, but there are also font sizes that
can be adjusted. There are key words that are used to adjust them:

  • xx-small

  • x-small

  • small

  • medium

  • large

  • x-large

  • xx-large

Font sizes can be changed using pixels and ems. ems is another way
to manipulate the size of the texts. All major browsers will resize the texts
for you. If you havent already specified the font size, the em can be calculated.
Knowing that the default browser size is 16px the following can calculate the em:

em=pixels/16

Font-Weight

Similar to font styles, there are three additional values for the wieght of the font, excluding the normal value. Here are all the font weights:

  • Normal

  • Lighter

  • Bold

  • Bolder

The font weight can also be defined with numbers
ranging from 100 (Thin) to 900 (thick). 400 is
similar to normal and 700 is similar to bold.

Text Decoration

There are five commonly used values to decorate text:

None

Defines text normally.

Inherit

Inherits the property from its parent element.

Overline

Draws a horizontal line above the text.

Underline

Draws a horizontal line below the text.

Overline

Draws a horizontal line above the text.

Line-Through

Draws a horizontal line through the text.

Another value is blink, however, most browsers ignore it

Letter Spacing

Letter spacing specifies the space between in characters

You can have positive values which increases the space between the characters.

Or you can have negative values that decreases the space between the characters.

Word Spacing

Similar to letter spacing, you can also specify the space between each word.

You can have positive values to increase the space betwee words.

You can have negative values to decrease the space betwee words.

White Spacing

White space property specifies how white space inside and element is handled. There are four values:

nowarp Puts the whole text onto one line and colapses all the whitespace into single whitespace.
pre Text will only wrap on line breaks and white space.
pre-line Text will wrap where there is a break in code, but extra white space is ignored.
pre-wrap Text will wrap when necessary, and on line breaks
