Note however, there are a few things that are needed to be taken into consideration when making shadows for texts:

  • When you place a shadow close to the text, it is usually not as blurred as when you place a shadow further away from the text.
  • When you place a shadow that is close, it implies one of the three things. Either the underlying surface is close, the light source is close or both the underlying surface and light source is close.
  • A shadow that is close usually is darker compared to a shadow that is distant because less light can get in between the shape and the underlying surface.

Psuedo Classes and Elements

In CSS3, Psuedo-Classes is a technique that allow us to style elements or just part of teh element, that exists within a document tree without using any scripts such as JavaScript. Psuedo-Elements are used to select specifc parts of an element. In order to use either Psuedo-Class or Psuedo-Element you use semi-colons. For the Psuedo-Class you use one semi-colon ":", with Psuedo-Element you use two "::".

There are two commonly used psuedo classes; there is the ":first-child" class, and there is the ":last-child" class. The ":first-child" class matches an element that is the first child element of some other element. The ":last-child" class matches an element that is the last child element of some other element.
