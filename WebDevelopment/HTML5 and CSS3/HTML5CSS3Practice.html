Differences between HTML with CSS and HTML5 with CSS3

The Header Element

The header element is a new way to define the header of a website. Normally you would use div followed by id then write header, but with HTML5 you can simply do that in the body element.

The Footer Element

The footer element is usually found at the bottom of the site. Stuff like contact details and other minor details can be found in the footer.

The Nav Element

Making Box Shadows

In CSS3 you can also add shadows to your box elements using the box shadow element. With this element, there is a specific order the values are placed in.

  • Horizonatl Offset
  • Vertical Offset
  • Blur
  • Spread
  • Colour

With the exception of the blur values, all the other values can be a negative value to give a shadow in the opposite direction. The two examples below will demonstrate how the box shadows will work.
